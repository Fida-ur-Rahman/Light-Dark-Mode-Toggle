How It Works

This theme switcher uses CSS custom properties (variables) to define color values for both light and dark themes. JavaScript toggles the dark-mode class on the document body, which changes all variable values. Your preference is saved in localStorage so it persists across visits. The design is fully responsive and works on all device sizes.
